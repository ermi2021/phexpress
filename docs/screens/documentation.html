Getting started

before you get started we need to identify all the necssery Requirements

  • PHP 8.1 or above should be installed
  • Composer Must be Installed (install Composer )
  • Mysql mustbe installed

Run the below command to create a new project

> composer create-project phexpress/phexpress awsome-project

After the project is created you can run the below command to navigate to the project and start the server

> cd awsome-project

> php -S localhost:3000

Now you can open your browser and navigate to localhost:3000 to see the default page
